# 🎮 Gestor de Inventario - Tienda de Videojuegos

Aplicación de consola en Java para gestionar el inventario de una tienda de videojuegos.
Permite añadir artículos, consultar stock y realizar ventas con control de excepciones.

---

## 📁 Estructura del proyecto

```
tienda-videojuegos/
├── src/tiendavideojuegos/
│   ├── Item.java
│   ├── Vendible.java
│   ├── StockAgotadoException.java
│   ├── Videojuego.java
│   ├── VideojuegoConDescuento.java
│   ├── Consola.java
│   ├── Inventario.java
│   └── Main.java
├── docs/        ← Javadoc generado
├── .gitignore
└── README.md
```

---

## ▶️ Compilación y ejecución

### Con NetBeans
Abre el proyecto y pulsa el botón **Run** (F6).

### Desde terminal
```bash
javac -d bin src/tiendavideojuegos/*.java
java -cp bin tiendavideojuegos.Main
```
